# [HAVENSHOT]

![Screenshot](/assets/screenshot-400x250.png)

A wellness-focused action game created by Dev Agnihotri for Mindhaven.

[HAVENSHOT] is intended to be a throwback at 90s first person shooters, reimagined as a therapeutic gaming experience that helps users release stress through focused action gameplay.

The game features 5 different weapons:

- Pistol
- Rocket Launcher
- Shotgun
- Machine Gun
- Rail Gun

The goal is to score as many points as possible by eliminating robots on the station with style.
Points are awarded for performing different feats, such as eliminating targets, making direct rocket hits, hitting while jumping, achieving combos.
